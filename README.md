<section class="main-section" id="A_linguagem_C" id="1"><header>A linguagem C</header>
<p>
 A linguagem C é uma das linguagens de programação mais clássicas e influentes do mundo. Criada na década de 1970, destaca-se por seu alto desempenho, controle direto sobre o hardware e gerenciamento de memória. 
</p> <p>Além de sua importância prática, a linguagem C teve grande influência no surgimento de outras linguagens de programação, como C++, Java e C#. Seu aprendizado ajuda a compreender conceitos fundamentais da programação, como estruturas de controle, funções, ponteiros e gerenciamento de memória.</p>

<section class="main-section" id="Variaveis" id="2"><header>Variaveis</header>
<p>Em C, variáveis são espaços na memória do computador usados para armazenar dados que podem ser alterados durante a execução do programa. Cada variável possui um tipo, que define o tipo de informação que ela pode armazenar, como números inteiros, números decimais ou caracteres.</p> <p>As variáveis são fundamentais na programação, pois permitem armazenar, manipular e processar informações durante a execução do programa. Elas são utilizadas em cálculos, tomadas de decisão, repetições e praticamente todas as operações realizadas por um software.</p>  
<code>let</code>
<ul>Principais Variavies (Reservadas)

  </ul>
